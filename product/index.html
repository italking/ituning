Overview

Product architecture

TMaestro is an intelligent parameter adjustment product that can help developers and operators automatically adjust the parameters of the application and analyze the reasonable configuration parameters of the JVM through intelligent algorithms. Hosting the entire parameter configuration execution process can achieve real-time large-scale cluster tuning of the online environment without personnel participation. Improve application performance and stability, reduce the use of physical resources of applications, and save user costs.

Differences with industry products

At present, the mainstream JVM tools in the industry mainly include APM vendors, open source free and commercial Jvm analysis tools, mainly for observable and alarm-related work, requiring technicians to manually tune according to the analysis results, TMaestro can analyze the results according to the algorithm, can be automatically tuned in the online environment, and has the industry’s leading automatic tuning technology capabilities.

Usage scenarios

At present, the application of Jvm tuning a lot of rely on manual experience, can not do online real-time tuning, can not carry out large-scale cluster real-time tuning, TMaestro can solve the above problems very well, reduce the difficulty of tuning and cost, to ensure the stability and performance of the application

Daily tuning

After TMaestro configures the tunable execution cycle, Tmaestro will tune the parameters of the online cluster in real time to ensure stable operation and high performance of the cluster.

App deployment tuning

After the application is released, the application tuning analysis is carried out to ensure the upgrade quality of the application cluster and avoid the occurrence of “degradation and upgrading”.

Product core competencies

Automatic metric collection

Monitoring metrics are collected through the Agent, and relevant monitoring metrics can be automatically collected by simply installing them

Intelligent tuning parameter analysis

Intelligent algorithm tuning parameter analysis, automatically analyze the effective tuning parameters based on historical monitoring data

Tune full-process hosting

TMaestro can perform batch parameter tuning for multiple application clusters, and the entire tuning process from parameter issuance to parameter effectiveness does not require personnel to participate。

Intelligent risk identification

With intelligent risk identification ability, automatic rollback when encountering risks, batch tuning parameters will automatically perform adjustments in batches
